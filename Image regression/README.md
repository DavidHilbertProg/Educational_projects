# В данном дериктории проект для соревнования на kaggle по определению численных значений компонентов биомассы по изображением растительности

- biomass_predict

  Краткое описание:
Использование предоубченной модели ResNet50 для изображений. Сравнение различных моделей: ElasticNet, Ансамбли, Lasso, Stacking, бустинги и выявление лучшей

- biomass_predict_2

  Краткое описание:
Попытка улучшить модель biomass_predict по полученным данным. Анализ распределения целевых переменных, статистических характеристик, корреляций между компонентами биомассы.
Извлечение дополнительных признаков из изображений, используя CV, добавление дополнительных спектральных индексов для изображений. Увеличение датасета с помощью бутстрапа.
Решение с помощью анасамбля нейросети и индивидуальных моделей для каждого компонента, что в конечном итоге помогло улучшить метрику R^2 с изначальных 0.54 до 0.92
