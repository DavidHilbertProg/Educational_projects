# Пет-проекты по кредитному скорингу

- credit_scoring

  Краткое описание:
Определение кредитного рейтинга клиентов в условиях несбалансированных классов. Применялась методика oversampling (SMOTE), чтобы сбалансировать классы.
Прогноз был выполнен с помощью моделей Ridge, CatBoost и нейросетевых MLP моделей с использованием PyTorch и Keras. Для моделей расчитаны метрики:
accuracy, f1-scroe, pressision, recall. Лучших результатов удалось добиться с помощью CatBoost

- bayesian_classifier

  Краткое описание:
  Определение кредитного рейтинга клиентов с использованием байесовского классификатора
