# Пет-проекты по определению цены на недвижимость

- random_forest

  Краткое описание:
Модель Random Forest с подбором количества деревьев по метрике MAE.

- dt_lr_boosting

  Краткое описание:
Определение цены на недвижимость используя различные модели: деревья решений, линейная регрессия с L-1 и L-2 регуляризацией, XGBboost, LightGBM, CatBoost,
с оценкой качества моделей по метрике SMAPE
